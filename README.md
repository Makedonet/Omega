# OMEGA — демо-лендинг (портфолио)

**Лайв-демо: https://omega-xi-seven.vercel.app/**

Одностраничный сайт-визитка для вымышленного частного детективного агентства. Работа из портфолио веб-разработчика: адаптивная вёрстка с нуля, без CMS и без сборщиков.

## Стек

- Чистый HTML5 / CSS3 (CSS-переменные, flex/grid, медиа-запросы)
- Vanilla JavaScript (без jQuery и фреймворков)
- [Swiper 11](https://swiperjs.com/) — карусель отзывов
- [Web3Forms](https://web3forms.com/) — приём заявок с формы (без бэкенда)
- Google Fonts (Montserrat)
- Хостинг: [Vercel](https://vercel.com/) (статический деплой)

## Что внутри

- Hero с фоновым изображением и анимацией появления
- Секция «О нас», 17 карточек услуг с hover-эффектом переворота
- Блок преимуществ, карусель отзывов
- Контакты, плавающий FAB-виджет мессенджеров
- Модальное окно формы заявки с валидацией и AJAX-отправкой
- Бургер-меню и адаптивные брейкпоинты под 1024 / 768 / 480 px

## Локальный запуск

Любой статический сервер из корня проекта:

```bash
python -m http.server 8000
# или Live Server в VS Code
```

Открыть `http://localhost:8000`.

## Деплой

Подключить репозиторий к Vercel — деплой статикой без билд-команды. `vercel.json` задаёт `cleanUrls`. Для рабочей формы — подставить свой `access_key` от [web3forms.com](https://web3forms.com/) в `index.html` (константа `WEB3FORMS_ACCESS_KEY`).

## Заметки по приватности

Все контактные данные (телефон, email, мессенджеры) и имена в отзывах — вымышленные. `robots.txt` и `<meta name="robots" content="noindex,nofollow">` запрещают индексацию, чтобы демо не конкурировало с оригинальным сайтом в выдаче.
