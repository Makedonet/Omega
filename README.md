# OMEGA — лендинг частного детективного агентства

[![Лайв-демо](https://img.shields.io/badge/Live_demo-omegadet.vercel.app-162CBD?style=for-the-badge)](https://omegadet.vercel.app/)

Одностраничный сайт-визитка с адаптивной вёрсткой, написанный с нуля без CMS, сборщиков и фреймворков. Демонстрационная работа из портфолио.

## Технологии

- **HTML5 / CSS3** — CSS-переменные, flex/grid, медиа-запросы, dvh/env() для iOS
- **Vanilla JavaScript** — без jQuery и фреймворков
- **Swiper 11** — карусель отзывов
- **Web3Forms** — приём заявок без своего бэкенда
- **Vercel** — статический хостинг

## Что внутри

- Hero с фоновым изображением и анимацией появления
- 17 карточек услуг с hover-эффектом переворота
- Карусель отзывов (Swiper) с навигацией и кастомным стилем
- Модальное окно формы заявки с валидацией и AJAX-отправкой
- Плавающий FAB-виджет мессенджеров
- Бургер-меню с translateX-анимацией
- Адаптивные брейкпоинты под 1024 / 768 / 480 px
- iOS Safari-оптимизации (URL-bar resize, safe-area-inset, tap-highlight, sticky-hover)
- A11y: `aria-label`, `role="dialog"`, корректные `alt`, закрытие модалок по Esc, `prefers-reduced-motion`

## Заметка

Контактные данные, имена в отзывах и название агентства — вымышленные. Сайт намеренно `noindex`, чтобы не пересекаться в выдаче с реальными организациями.
